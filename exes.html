LS-DYNA Version Control:

Dev version:
The version developers work on everyday. Contains the latest bug fixs and new features.
Branch version:
The version branched out to prepare for Release. For example, R9 is branched out to prepare for releases R9.0.1, R9.1.0 ... It contains the latest bug fixs but most time no new features.
Release version:
Release version branched out from "Branch version". Like R9.0.1 branched out from R9. It contains bug fixs until the "cut-off" day.
The number:
The number is an unique tag for a specific SVN version. Larger the number, later the version. Please note, dev, branch and release all share the same set of numbers.

Known Problems

S-ALE timestep drop, mysterious internal energy increase, most likely happen in blast problems.
Fixed in dev.114999 and R9.115000 and R10.115001 on March 04, 2017. Fix NOT available for R9.0.1 and R9.1.0 releases.
ESORT=1 with *INITIAL_VOLUME_FRACTION_GEOMETRY, MPP only. Initial volume fraction might be wrong.
Fixed in dev.121441+dev.121494 and R10.121444+R10.121499. Please use versions later than the above mentioned.
General MPP memory bug. Unpredictable behavior.
Avoid using versions between dev.122499 and dev.123354 (DEV only, R9, R10 not affected)
